# TenTric — Automated Sign-In

A minimal automation tool that handles the daily TenTric sign-in flow. Built to remove repetitive steps and provide a clean, reliable one-command experience.

## Overview

TenTric’s sign-in process is straightforward but repetitive.  
This script automates the entire workflow using Selenium — launching the browser, logging in, and completing the sign-in action with consistent reliability.

## Features

- Automated login + sign-in sequence  
- Simple, readable code structure  
- Selenium-based browser automation  
- Clear logging for each run  
- Easy to extend or schedule
